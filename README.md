# 🚀 Portfólio - Blazor WebAssembly

Este projeto é um site de portfólio desenvolvido com **C#** e **Blazor WebAssembly**.

## 📌 Sobre o projeto

O objetivo deste projeto é servir como um portfólio pessoal, apresentando minhas habilidades, projetos e experiências na área de desenvolvimento.

A base do projeto foi construída a partir de um template existente, que foi adaptado e customizado com minhas próprias informações, identidade visual e estrutura.

## ⚙️ Tecnologias utilizadas

- C#
- Blazor WebAssembly
- HTML / CSS
- .NET

## 🚧 Em desenvolvimento

Algumas funcionalidades ainda estão em construção, incluindo:

- Sistema de autenticação
- Login de usuários

Essas funcionalidades ainda não estão visíveis na interface final, pois estão sendo desenvolvidas gradualmente.

## 🌐 Deploy

O projeto está hospedado utilizando um subdomínio gratuito fornecido por **:contentReference[oaicite:0]{index=0}**.

## 📎 Objetivo

Além de servir como portfólio, este projeto também é utilizado como laboratório para aplicar conceitos como:

- Arquitetura de software
- Boas práticas de desenvolvimento
- Evolução contínua de features
